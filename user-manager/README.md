# 用户管理小工具

独立于主程序的本地工具，用于添加、删除、编辑用户及权限，直接读写与主程序相同的 `users.json`。

## 使用方式

1. **双击运行**：双击 `启动用户管理.bat`，会自动打开浏览器。
2. **命令行**：在本目录执行 `node server.cjs`，也会自动打开浏览器。

页面中可：
- 查看用户列表（用户名、角色、权限）
- 添加用户（用户名、密码、角色、权限勾选）
- 编辑用户（改密码、角色、权限，密码留空表示不修改）
- 删除用户（不能删除最后一名管理员）

修改后保存即写入 `users.json`，主程序无需重启，用户重新登录后生效。

## 配置（可选）

若你的资源根目录不是 `D:\尚品易站图片`，可复制 `config.json.example` 为 `config.json`，并修改其中的 `assetsRoot` 为你的路径。例如网盘盘符或主程序使用的资源目录。

未配置时默认使用 `D:\尚品易站图片`，且会在此路径下的 `程序图片勿动\users.json` 读写用户数据（与主程序一致）。
